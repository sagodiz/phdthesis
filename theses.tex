\noindent\textbf{Declaration}
\vspace*{0.5cm}

In the PhD dissertation of Zoltán Ságodi entitled ,,\dolgozatcim'', Zoltán Ságodi's contribution was decisive in the following results:

\begin{enumerate}[label=Chapter~\arabic*:,
	leftmargin=!,
	labelwidth=!,
	align=left]
	\setcounter{enumi}{2}
	
	\item \textbf{Zoltán Ságodi}, István Siket, and Rudolf Ferenc\\
	\newblock {Methodology for Code Synthesis Evaluation of LLMs Presented by a Case Study of ChatGPT and Copilot}.
	\newblock \emph{IEEE Access}, VOL(12), 72303-72316, 2024.
	\begin{itemize}
		\item I collected relating methodologies and determined the gaps in those methodologies.
		
		\item I created the comparison methodology.
		
		\item I searched for an evaluation benchmark.
		
		\item I evaluated the Language Models on the benchmark.
		
		\item I performed the quality analysis of the synthetized programs.
		
		\item I designed the human evaluation.
		
		\item I created the Human Evaluation framework.
	\end{itemize}
	
	\item \textbf{Zoltán Ságodi}, István Kolláth, Péter Heged\H{u}s and Rudolf Ferenc
	\newblock {A Program Synthesis Dataset for LLM Temperature Analysis}.
	\newblock \emph{IEEE Access}, VOL(13), 184022-184029, 2025.
	\begin{itemize}
		\item I designed the study.
		
		\item I created the model selection framework.
		
		\item I created the model evaluation framework.
		
		\item I evaluated the data according to the temperature variable effect.
		
	\end{itemize}
	
	\item \textbf{Zoltán Ságodi}, Gábor Antal, Bence Bogenfürst, Martin Isztin, Péter Heged\H{u}s and Rudolf Ferenc
	\newblock {Reality Check: Assessing GPT-4 in Fixing Real-World Software Vulnerabilities}.
	\newblock In \emph{Proceedings of the 28th International Conference on Evaluation and Assessment in Software Engineering (EASE '24)}, Association for Computing Machinery, 252–261, 2024.
	\begin{itemize}
		\item I searched for prompt engineering techniques.
		
		\item I designed the evaluation of the prompts and also the models.
		
		\item I participated in the manual evaluation of the generated source code.
		
		\item I participated in the manual evaluation of the generated textual responses.
	\end{itemize}
\end{enumerate}

These results cannot be used to obtain an academic research degree, other than the submitted PhD thesis of Zoltán Ságodi.
